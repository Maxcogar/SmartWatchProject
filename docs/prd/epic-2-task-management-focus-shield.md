# Epic 2: Task Management & The Focus Shield

* **Story 2.1: Bluetooth Connectivity & Handshake**
  * **AC:** Watch and phone can establish a stable BLE connection and exchange a basic message.
* **Story 2.2: Task Synchronization and Display**
  * **AC:** The watch can request, receive, and display a scrollable list of tasks from the phone.
* **Story 2.3: The Active Task Screen UI**
  * **AC:** Tapping a task opens the split-screen UI with the task on one side and timer controls on the other.
* **Story 2.4: Functional Focus Timer**
  * **AC:** The timer can be started, paused, and resumed. It enters an always-on state when active and triggers a clear alarm when finished.
* **Story 2.5: The Focus Shield Implementation**
  * **AC:** When the Active Task Screen is visible, all incoming notifications are queued in memory and are not displayed.
