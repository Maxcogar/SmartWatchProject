# Epic 3: External Connectivity & Notifications

* **Story 3.1: Queued Notification Display**
  * **AC:** After leaving the Active Task Screen, queued notifications are displayed one by one and must be swiped to be dismissed.
* **Story 3.2: Task Completion Sync**
  * **AC:** A "Complete" button on the Active Task Screen sends a signal to the phone, which then marks the task as complete in Microsoft ToDo.
* **Story 3.3: Modular Control Panel UI & Gesture**
  * **AC:** An edge-swipe from the right reveals the translucent control panel overlay. An edge-swipe from the left dismisses it.
* **Story 3.4: Network Device Control Implementation**
  * **AC:** The "Compressor" button in the control panel successfully sends the correct HTTP requests to the predefined local IP addresses.
